# DCS-WABU
DCS-WABU (Weight and Balance Utility)

This is a calculator and exporter for Digital Combat Simulator (DCS).

Weight and Balance implemented for: 

-A10C

-F-14B

-F15C

-F18C

-Bf 109

-Su-33

-Viggen

-M2000C Mirage

-AV-8B Harrier

-A-4E-C

-Huey

Loadout export enabled for the F18C.
